         PHX MCP - QUICK FIX untuk "No Tools" Issue
═══════════════════════════════════════════════════════════════

MASALAH: Server berjalan tapi tools tidak muncul di Cursor

SOLUSI: Copy config ini ke Cursor settings!

═══════════════════════════════════════════════════════════════
LANGKAH 1: Buka Cursor Settings
═══════════════════════════════════════════════════════════════

  1. Tekan: Ctrl + Shift + P
  2. Ketik: "Preferences: Open User Settings (JSON)"
  3. File settings.json akan terbuka

═══════════════════════════════════════════════════════════════
LANGKAH 2: Copy-Paste Config Ini
═══════════════════════════════════════════════════════════════

{
  "mcpServers": {
    "phx-ai": {
      "command": "wsl",
      "args": [
        "--",
        "/mnt/d/aihex/PHX/hexstrike-wsl-env/bin/python3",
        "/mnt/d/aihex/PHX/phx_mcp.py",
        "--server",
        "http://10.255.255.254:8888"
      ],
      "description": "PHX AI MCP - Cybersecurity Tools",
      "timeout": 300,
      "disabled": false
    }
  }
}

═══════════════════════════════════════════════════════════════
LANGKAH 3: Restart Cursor
═══════════════════════════════════════════════════════════════

  Tutup Cursor sepenuhnya dan buka kembali

═══════════════════════════════════════════════════════════════
LANGKAH 4: Test
═══════════════════════════════════════════════════════════════

  Di Cursor chat, ketik:
  
  "list all available MCP tools"

  Atau:
  
  "use PHX tools to check system"

═══════════════════════════════════════════════════════════════
CATATAN PENTING
═══════════════════════════════════════════════════════════════

✅ Server PHX sudah berjalan di port 8888
✅ Config sudah diperbaiki dengan path dan IP yang benar
✅ WSL connectivity sudah verified

Jika IP 10.255.255.254 tidak work, cek dengan:
  
  wsl cat /etc/resolv.conf

Lalu update IP di config.
